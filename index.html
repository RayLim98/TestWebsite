HTML and CSS

Tags

Tags are used to declare the following content to inherit some form of meaning and information. Tags are used to either give certain words or characters a semetic value, or simply used stylistically. The use of bold or italics will help the viewers see meaning in text. The use of headings and paragraphs are pretty straight forwards so I don't think I'll need to brush over this.

  • ⟨ b ⟩ for bold
  • ⟨ i ⟩ for italics
  • ⟨ strong ⟩ to define a important text
  • ⟨ em ⟩ for emphasized text
  • ⟨ h1,2,3,4... ⟩ for headings
  • ⟨ p ⟩ for paragraphs
  • ⟨ a ⟩ for hyperlink

Attributes

Attributes are further declarations that are added along side tags. They section, style and alter content. There are so many atributes, so the ones that will be looked into this section will be for sectioning and images; as it provides good practices when it comes to using CSS.

  • Class
  • Id
  • src
  • href
